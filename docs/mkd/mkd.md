## 折叠图片：[原文链接](https://www.cnblogs.com/cnblogswilliam/p/14448830.html)

### 格式如下 



```html

<details>
	<summary>标题</summary>
	<p>
		内容
	</p>
</details>

内容 = 普通文字：直接输入文本即可

内容 = code ：格式为 <code>代码</code>

内容 = 图片 ：<img src="路径"/> 

（需要缩放时 形如：<img src="" style="90%"/>）

```
